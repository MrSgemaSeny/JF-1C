## Скриншот 2 — Карточка сделки

**Воронка продаж (стадии сделки):**
- Новая → Подготовка документов → Счёт на предоплату → В работе → Финальный счёт
- Визуально — горизонтальный прогресс-бар со стадиями, текущая подсвечена

**Поля сделки (левая панель):**
- Стадия (текущая)
- Сумма и валюта (500 тенге)
- Кнопка "Принять оплату"
- Блок "Оплата и доставка" (с возможностью добавить)
- Итого к оплате по сделке
- Дата завершения
- Клиент + Контакт (имя, телефон, email)
- Адрес доставки

**Табы внутри сделки:**
- Общие / Товары / Предложения / Роботы / Счета / Связи / История / Маркетплейс / Ещё

**Правая панель (активности/таймлайн):**
- Дело / Комментарий / Сообщение / Онлайн-запись
- Поле "Что нужно сделать" (создание задачи/дела)
- "Пригласить к обсуждению"
- Кнопка "Связаться с клиентом" (автодело с дедлайном)
- Дедлайн дела с уведомлением (колокольчик)
- Кнопки "Выполнено" / "Редактировать" у дел

**Верхняя панель действий:**
- Звонок / Email (иконки быстрого контакта)
- Расширения
- Документ (выпадашка)
- Предложение (CTA-кнопка)

**Выпадашка "Документ" — типы документов (Казахстан):**
- Акт
- Счёт
- Доверенность
- Счёт-фактура
- Накладная
- Коммерческое предложение
- Список документов
- Добавить новый шаблон
- Расширения

## Скриншот 4 — Канбан сделок

**Канбан-доска:**
- Переключение вида: Канбан / Список / Дела / Календарь
- Фильтры: Входящие / Запланированные / Ещё
- Фильтры по ответственному и статусу дел (Просрочены, Пора...)
- Выбор воронки ("Общая воронка")
- Кнопка "+ Создать" (сделку)

**Карточка сделки на канбане:**
- Название сделки
- Сумма
- Клиент (контакт — "Без имени")
- Время ("6 минут назад")
- Иконки: звонок, email, чекбокс
- "+ Дело" — быстрое добавление активности
- Таймер/счётчик времени
- Аватар ответственного

**Стадии воронки (колонки):**
- Новая / Подготовка документов / Счёт на предоплату / В работе / Финальный счёт / Сделка провалена
- У каждой — счётчик сделок и сумма

**Дополнительно (скрин 3, левая панель):**
- Тип сделки: Продажа
- Дата начала
- Доступна для всех: Да
- Ответственный (с аватаром)
- Итог сделки

## Итоговый список фич для твоего проекта

Разбил по категориям:

### Сделки (Deals)
- Создание сделки с суммой и валютой
- Стадии (кастомная воронка)
- Поля: клиент, контакт, дата, адрес, ответственный, тип, доступность
- Привязка товаров к сделке
- Привязка документов (счёт, акт, накладная и т.д.)
- Кнопка "Принять оплату"
- История изменений

### Активности (Activities / Дела)
- Создание дел внутри сделки (звонок / встреча / задача / письмо)
- Дедлайн + уведомление
- Статус: Выполнено / В работе
- Быстрое добавление прямо с канбана

### Канбан
- Drag & drop по стадиям (предполагается)
- Счётчик и сумма по каждой стадии
- Быстрые действия с карточки (звонок, email, дело)
- Несколько воронок

### Документы
- Генерация документов по шаблону (казахстанские форматы)
- Кастомные шаблоны

### Задачи
- Отдельный модуль задач (не путать с делами в сделке)
- Множество видов: список, гант, календарь, план
- Фильтрация по роли, статусу, проекту
- Постановщик / Исполнитель / Проект

### Авторизация
- OAuth (Google)
- Email + пароль
