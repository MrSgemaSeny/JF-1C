# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover any security vulnerabilities, please do not open a public issue. Instead, report them via one of the following methods:

* **GitHub Security Advisory:** Submit a report directly through the "Security" tab -> "Advisories" -> "Report a vulnerability" of this repository.
* **Email:** Send a detailed description to: security@yourdomain.com

### What to expect

* **Acknowledgment:** You will receive a response within 48 hours acknowledging your report.
* **Status Updates:** We will provide updates every 3–5 business days regarding the progress of the fix.
* **Disclosure:** Once a fix is verified and deployed, a public Security Advisory will be published with credit to the reporter (if desired).
