# Бэкенд — Ядро Задач (Tasks)

Управляет жизненным циклом карточек на [[12_Фронтенд_ЛичныйКабинет_Сотрудник]].

## Зоны ответственности
- **TaskController**: REST API для CRUD операций с задачами.
- **TaskBatchUpdate**: Возможность массово менять статус задач (например, при перетаскивании всей колонки).
- **Комментарии и Документы**: Привязка файлов и сообщений к конкретной `TaskEntity`.

Данные хранятся в [[30_Инфраструктура_И_БД]] (PostgreSQL).
