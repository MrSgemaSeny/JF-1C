# Фронтенд — Авторизация

Экраны доступа к приложению.

## Зоны ответственности
- **LoginPage**: Вход по Email и Google.
- **RegisterPage**: Регистрация с валидацией сложности пароля. Выбор роли (Client / Employee).
- **CompleteProfilePage**: Окно дозаполнения данных (номер телефона, название компании) для новых клиентов, авторизовавшихся через Google.
